# Disciplina-Desenvolver-e-organizar-interface-grafica-para-aplicacoes-desktop

Sistema de gerenciamento de biblioteca digital desenvolvido em Java com interface gráfica Swing.

Sobre o Projeto
O Sistema de Biblioteca Digital é uma aplicação que permite o armazenamento e gerenciamento de livros digitais. 
Administradores podem cadastrar e organizar o acervo, enquanto usuários podem explorar, favoritar e gerenciar suas leituras.
